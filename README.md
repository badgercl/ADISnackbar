# ADISnackbar
Nicer Snackbars 

We'll be adding more functionality as we need it in out project ;)


## Install

Add it in your root build.gradle at the end of repositories:

```gradle
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

Add the dependency
```gradle
	dependencies {
	        compile 'com.github.badgercl:ADISnackbar:v0.1'
	}
```
